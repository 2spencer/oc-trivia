# OC Trivia Nights Guide

A responsive web guide to pub trivia nights happening across South Orange County venues.

## Features

- **List View** - Browse all upcoming trivia events in a clean grid layout
- **Calendar View** - See events overlaid on a monthly calendar
- **Day Filtering** - Filter trivia nights by day of the week
- **Venue Links** - Direct links to Instagram and websites for each venue

## Venues

Browse trivia nights at:
- High Tide Coffee (Laguna Niguel)
- OC Local Taproom
- Selma's Chicago Pizzeria
- Wigz Sandwich Shop
- Los Molinos Beer
- And more

## Getting Started

Just open `index.html` in your browser. No installation or build process required.

## Deployment

Deployed to Vercel with automatic updates on every GitHub push.
